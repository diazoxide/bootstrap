# Installation

```shell
wget -O /usr/local/lib/bs.zip https://github.com/diazoxide/bootstrap/archive/refs/heads/master.zip && rm -rf /usr/lib/bs && unzip -q /usr/lib/bs.zip -d /usr/lib/bs && chmod +x /usr/lib/bs/bootstrap-master/bs.py && rm /usr/local/bin/bs && ln -s /usr/lib/bs/bootstrap-master/bs.py /usr/local/bin/bs 
```

# Usage

```shell
bs help
```
